# WiiQuizz

Ce programme vous permet d'animer vos soirées entre amis en utilisant les manettes de Wii comme 
contrôleurs de différents jeux de soirée comme :

- Buzzer Simple
- Devinez l'image
- Blind Test

et encore d'autres à venir !
